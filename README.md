# 📅 Schedulrr – A Calendly Clone

**Schedulrr** is a full-featured calendar scheduling platform inspired by [Calendly](https://calendly.com/), built with modern technologies including Next.js, TypeScript, Clerk authentication, Tailwind CSS, and Shadcn UI.

---

## 🚀 Features

- 🔐 **User Authentication** – Secure sign-up and login with [Clerk](https://clerk.dev)
- 📆 **Event Types** – Create custom meeting types (15 min, 30 min, etc.)
- 🕒 **Availability Management** – Set working hours and time zones
- 📩 **Booking Flow** – Shareable booking links, auto time slot generation
- 🎨 **Modern UI** – Built with Tailwind CSS and [Shadcn UI](https://ui.shadcn.com/)
- ⚙️ **Server Actions** – Powerful backend logic using Next.js 15 app router
- ☁️ **Deployment** – Live on [Vercel](https://vercel.com)
